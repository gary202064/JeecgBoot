# 工人工序熟练度功能实现说明

## 业务需求
将工人熟练度从单一字段改为工序维度的熟练度，一个工人可以对多个工序有不同的熟练度等级。

## 数据库变更

### 1. hn_worker_process_ability 表（工人-工序能力表）
**新增字段：**
- `skill_level` VARCHAR(32) - 熟练度（使用字典 skill_level）

**SQL语句：**
```sql
ALTER TABLE `hn_worker_process_ability` 
ADD COLUMN `skill_level` VARCHAR(32) NULL COMMENT '熟练度' AFTER `can_full_work`;
```

### 2. hn_worker 表（工人表）
**删除字段：**
- `skill_level` - 原有的单一熟练度字段

**SQL语句：**
```sql
ALTER TABLE `hn_worker` 
DROP COLUMN `skill_level`;
```

## 代码变更

### 1. 实体类变更

#### HnWorkerProcessAbility.java
**新增字段：**
```java
/**熟练度*/
@Excel(name = "熟练度", width = 15, dicCode = "skill_level")
@Schema(description = "熟练度")
@Dict(dicCode = "skill_level")
private java.lang.String skillLevel;
```

#### HnWorker.java
**删除字段：**
- 删除原有的 `skillLevel` 字段及其注解

### 2. Service 层变更

#### IHnWorkerService.java
**新增方法：**
```java
List<HnWorkerProcessAbility> getWorkerProcessAbilities(Long workerId);
```

#### HnWorkerServiceImpl.java
**实现方法：**
```java
@Autowired
private IHnWorkerProcessAbilityService hnWorkerProcessAbilityService;

@Override
public List<HnWorkerProcessAbility> getWorkerProcessAbilities(Long workerId) {
    LambdaQueryWrapper<HnWorkerProcessAbility> queryWrapper = new LambdaQueryWrapper<>();
    queryWrapper.eq(HnWorkerProcessAbility::getWorkerId, workerId);
    return hnWorkerProcessAbilityService.list(queryWrapper);
}
```

### 3. Controller 层变更

#### HnWorkerController.java
**新增接口：**
```java
@Operation(summary = "工人表-获取工人工序熟练度列表")
@GetMapping(value = "/getProcessAbilities")
public Result<List<HnWorkerProcessAbility>> getProcessAbilities(@RequestParam(name="workerId",required=true) Long workerId) {
    List<HnWorkerProcessAbility> abilities = hnWorkerService.getWorkerProcessAbilities(workerId);
    return Result.OK(abilities);
}
```

## 字典配置

需要在系统字典中配置 `skill_level` 字典，建议值：
- 初级
- 中级
- 高级
- 专家级

## API 接口说明

### 获取工人工序熟练度列表
- **接口路径：** `GET /hnworkerwage/hnWorker/getProcessAbilities`
- **请求参数：**
  - `workerId` (Long) - 工人ID（必填）
- **返回结果：**
  ```json
  {
    "success": true,
    "message": "操作成功！",
    "code": 200,
  ```

### 工人-工序能力管理接口
- **分页列表查询：** `GET /hnworkerwage/hnWorkerProcessAbility/list`
- **添加：** `POST /hnworkerwage/hnWorkerProcessAbility/add`
- **编辑：** `PUT /hnworkerwage/hnWorkerProcessAbility/edit`
- **删除：** `DELETE /hnworkerwage/hnWorkerProcessAbility/delete`
- **批量删除：** `DELETE /hnworkerwage/hnWorkerProcessAbility/deleteBatch`
